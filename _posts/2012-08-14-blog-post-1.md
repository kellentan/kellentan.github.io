Linear Algebra
======

Let's recall some key concepts that may prove useful from linear algebra. The span of a set of a vectors is the set of all points reachable via a linear combination of these vectors, i.e. $$\{x \;:\; x = \sum_{i} c_iv^{(i)} \;,\; v^{(i)} \in \{v^{(1)}, \cdots, v^{(m)}\}\}$$. Additionally, note that the span of the columns of a matrix $A$ can also be denoted as $$\text{range}(A)$$. For a linear system $$Ax = b$$, we say that a solution exists if $$b \in \text{range}(A)$$. A basis is linearly independent if each vector in the basis cannot be written as a linear combination of the other basis vectors. A singular matrix $A$ is one with linearly dependent columns. An orthogonal matrix is a square matrix with orthonormal (linearly independent with unit norm)rows and columns. These matrices have the nice property that $$A^{-1} = A^T$$.

Note the general definition of a norm, which must satisfy three basis properties:

- $$f(x) = 0 \;\rightarrow\; x = 0$$
- $$f(x + y) \leq f(x) + f(y)$$, aka the triangle inequality
- $$\forall \alpha \in \mathbb{R},\; f(\alpha x) = |\alpha|f(x)$$

Obviously, in ML we may only care about some of the more canonical norms. Another handy norm is the infinity norm $\lVert x \rVert_{\infty} = \max_i \|x_i\|$, i.e. simply the element with the largest absolute value. For matrices, we have the analogous Frobenius norm $$|lVert A |rVert\_{F} = \sqrt{\sum_{i, j} A_{i, j}^2} = \sqrt{\text{Tr}(A A^T)}$$, which can also be rewritten in terms of the trace operator.

With various datasets, we are particularly interested in matrix decompositions, which will help us with data storage/compression, and potentially noise reduction. There are two main avenues that are of interest. First, we write the eigendecomposition of a matrix $$A = V \text{diag}(\lambda) V^T$$, where V is a matrix where the columns are linearly independent eigenvectors, and $\lambda$ are the corresponding eigenvalues. This decomposition is particularly useful for examining the eigenvalues of a matrix: all positive eigenvalues necessitate that our matrix $A$ is positive definite, while all non-negative eigenvalues necessitate $A$ is positive semi-definite. Note that the existence of $\lambda_i = 0$ results in $A$ being singular, i.e. not invertible. Since the eigendecomposition may not necessarily exist, our more general technique is the Singular Value Decomposition (SVD), where $$A = UDV^T,\; A \in \mathbb{R}^{m \times n},\; U \in \mathbb{R}^{m \times m},\; D \in \mathbb{R}^{m \times n},\; V \in \mathbb{n \times n}$$. Note that $U, V$ are orthogonal matrices representing the left and right singular vectors, respectively, while $D$ is a diagonal matrix with the singular values. Interestingly, we note that the columns of $U, V$ are the eigenvectors of $AA^T$ and $A^T A$, respectively, and the non-zero singular values are the square roots of the eigenvalues of $A^T A$. 

A key application of matrix decomposition is in Principal Components Analysis (PCA), where our objective is to compress the given data into a more succinct representation. Let's assume we are given a dataset of points $$\{x^{(1)}, \cdots, x^{(m)}\}$$ s.t. each point $$x^{(i)} \in \mathbb{R}^n$$. We can encode each data point as $$f(x) = c = D^T x$$, and decode each data point as $$g(c) = Dc \approx x$$, where $$D \in \mathbb{n \times \ell}$$, and $\ell < n$. The fact $$c = D^T x$$ arises from solving $$c^{*} = \text{argmin}_{c} \lVert x - g(c) \rVert_{2}^{2}$$. Intuitively, $D$ is given by the $\ell$ eigenvectors corresponding to the $\ell$ largest eigenvalues (these are the principal components we would like to preserve in our compressed representation).
