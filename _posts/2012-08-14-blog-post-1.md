Linear Algebra
======

Let's recall some key concepts that may prove useful from linear algebra. The span of a set of a vectors is the set of all points reachable via a linear combination of these vectors, i.e. $$\{x \;:\; x = \sum_{i} c_iv^{(i)} \;,\; v^{(i)} \in \{v^{(1)}, \cdots, v^{(m)}\}\}$$. Additionally, note that the span of the columns of a matrix $A$ can also be denoted as $$\text{range}(A)$$. For a linear system $$Ax = b$$, we say that a solution exists if $$b \in \text{range}(A)$$. A basis is linearly independent if each vector in the basis cannot be written as a linear combination of the other basis vectors. A singular matrix $A$ is one with linearly dependent columns.
